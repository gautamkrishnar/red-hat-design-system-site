About this website

We create Red Hat digital experiences

Our teams work across the world in a range of disciplines, from UX research and digital design to web development and content strategy. We collaborate to make every digital interaction with Red Hat® reflect our core brand traits—open, authentic, helpful, and brave.

Our approach

Red Hat brand standards

Red Hat brand standards are the source code for our identity. They govern how we look and sound in all types of media. We follow brand standards to unify Red Hat digital experiences and stay up-to-date with our brand as it grows, improves, and adapts to meet new challenges.

PatternFly collaboration

We collaborate with PatternFly, an open source design system for enterprise product experiences. Using PatternFly as a foundation we create intuitive and scalable Red Hat digital experiences—the open source way.

Shared design language and vision

We share typography, scale, colors, and UI elements with PatternFly, so designers no longer need to reinvent the wheel when choosing a component or pattern to use. For example, an Accordion component used in a Red Hat enterprise web application will look the same when used on our websites.

Web components

Our team is building an open source web component driven development system to build scalable UI elements. This open source community project supports many components in our design system.

Design system fundamentals

Flexible

Our components can be arranged in a number of ways and CSS variables can be used for further customization.

Accessible

A goal of our design system is to create open, meaningful experiences that work for everyone, regardless of ability.

Consistent

Our documentation and tools streamline collaboration so that teams can create consistent, on-brand experiences.

Scalable

Our system enables teams to work concurrently across Red Hat's system of websites and beyond.
