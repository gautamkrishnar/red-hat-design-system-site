Default avatar with text - NJK

Overview

A Default avatar is a placeholder graphic for a photo or an image that’s placed to the left or on top of text.

Style

Default avatars can be used in light and dark themes. They feature a circular silhouette icon and are always grouped with text, like Attribution or some kind of descriptor.

Size

The size of Default avatars and text depends on where they’re used and how much information needs to be included.

For example, the size of Default avatars and text used in the navigation needs to be smaller than Default avatars and text used in layouts with more space. The maximum size of a Default avatar is 64px, so be mindful of the amount of text that’s included next to it.

Text positioning

Text options are positioned to the right or centered below a Default avatar.

Attribution or descriptor text is vertically-aligned vs horizontally-aligned.

Usage

Default avatars are subtle placeholder graphics used in the absence of photos or images. When a user doesn’t provide their own photo or image, a Default avatar is used instead.

Text

Text options can be placed under or to the right of a Default avatar.

For example, someone’s name can be placed on one line and their job information on the second line. Always group a Default avatar with Attribution or some kind of descriptor text. Never use it by itself.

Layout

Default avatars with text can be used in containers or layouts. They can be stacked horizontally or vertically depending on the amount of people and the container size.

Best practices

Default avatars shouldn’t be used without text.

Don’t use a dark theme Default avatar in the light theme.

Don’t include someone’s job information on the same line as their name, break it to the second line.

Behavior

Vertical vs. top alignment

If the height of text exceeds the height of a Default avatar, it becomes aligned to the top edge instead.

Attribution or descriptor text vertically-aligned vs. aligned to the top edge.

Responsive design

Default avatars with text can be used on desktop and mobile. If text that’s positioned on the right becomes too compressed, it will flow under and its alignment will change to centered. On mobile, Default avatars with text always stack vertically.

Spacing

Default avatars and text use PatternFly 4 spacers to define spacing values between elements.
