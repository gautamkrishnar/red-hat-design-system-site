Components

Overview

Components are interactive building blocks of our design system. Each component was created to meet a specific UI and accessibility need. Components should be used harmoniously together in the same space to create more intuitive user experiences.

Accordion

Toggles the visibility of multiple sections of content

Alert

Communicates a change in status or other information

Announcement banner

Announcement an important message across websites

Audio player

Plays and controls audio clips in a web browser

Avatar

A placeholder graphic for a photo or image

Block quote

A styled customer quote with attribution

Button

Allows a user to perform an action when triggered

Call to action

Directs a user to other pages or displays extra content

Card

Gives a preview of information in a small layout

Disclosure

Toggles the visibility of one section of content

Filter

Sorts a data set by predetermined sections and tags

Form

Collects a specific amount of information from a user

Jump links

Moves a user to content when a link is selected

Link with icon

An icon next to a link that adds extra context

Modal

Displays information or helps a user focus on a task

Navigation

Helps a user move seamlessly through a website

Promo banner

Advertises a promotional message or offer

Search bar

Anchored to the bottom edge of a page as content scrolls

Skip navigation

Jumps a user down to the main section of content

Sticky banner

Anchored to the bottom edge of a page as content scrolls

Sticky card

Anchored to the right edge of a page as content scrolls

Tabs

Organizes content in a contained view on the same page

Tooltip

Provides helpful information when hovered or focused

Video thumbnail

Indicates that a video can be played inline or in a modal
