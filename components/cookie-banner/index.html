Cookie banner

Overview

A Cookie banner is a notification banner that appears at the bottom of a Red Hat website indicating that Red Hat collects cookie data from a user.

Sample component

[cookie banner image]

Component status

Style

A cookie banner can be used in light and dark themes. It features legal text, a close button, and a very slightly transparent background container.

Theme

Close button

The close button is always placed in the top right corner of a cookie banner for familiarity, similar to other components.

Usage

A cookie banner informs visitors that the website they’re browsing uses cookie data. A cookie banner is required to be displayed for all users that have not previously closed the banner on other pages of the website. Cookie banner rules are standardized per country and must meet all regulations for cookie compliance. Rules of using the cookie banner exist outside of the design system and should be managed by a third party.

Grid

A cookie banner doesn’t fall within the grid, it spans almost the entire width of a browser window on any screen size.

Layout

A cookie banner contains legal information that should be placed on the bottom of a page so it doesn’t take away from the more important content on top. It’s still important enough to warrant persistence in order to ensure that visitors see it.

Best practices

Don’t use a light gray cookie banner in the light theme.

Don't replace the legal content, use the Announcement banner or the Persistent banner components for other content needs instead.

Don't omit the close button, it’s needed for accessibility.

Behavior

Overlay

A cookie banner will overlay the bottom of the page and content will scroll underneath it until it’s dismissed by a user.

Responsive design

A cookie banner can be used on smaller screens, but the limited screen space makes the banner taller.

Desktop

Tablet

Mobile

Interaction states

The only interactive element in a cookie banner is the close button. For more information about button interaction states, see Button.

Spacing

A cookie banner uses PatternFly 4 spacers to define spacing values between elements.
