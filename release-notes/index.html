Release notes

Overview

We're continually improving the Red Hat design system. The following changes are listed by the date we completed each change. The changes will be publicly available in the next major release following the listed date.

Design kit v1.6.0

Released on October 6, 2020.

Added

Avatar

  • Added dark theme Default avatar
  • Added dark theme Default avatar with attribution text

Button

  • Added dark theme Default avatar
  • Added dark theme Default avatar with attribution text

Color

  • Added dark theme Danger color (#e74c3c)

Form

  • Added light and dark theme Dynamic form
  • Added dark theme Standard form
  • Added dark theme input groups for Standard form

Label

  • Added filled and outlined Labels

Sticky card

  • Added light and dark theme Sticky cards as part of collecting Personalization patterns

Tooltip

  • Added dark Tooltips in top, right, bottom, and left orientations
  • Added drop shadow to dark Tooltip

Typography

  • Added dark theme text styles for desktop
  • Added dark theme list styles for desktop

Changed

Blockquote

  • Moved light and dark theme quote text below quote icon in all left-aligned configurations

Color

  • Changed dark theme link color for better accessibility

Interaction states

  • Changed color of focus indicator to stay current with browsers

Naming

  • Changed Form to Standard form

Tabs - Box

  • Changed styling of tab group to include inset on left

Deprecated

Global

  • Deleted Delete me empty layers in all components

Accordion

  • Deleted light and dark theme Accordions for mobile because they’re redundant

Alert

  • Deleted Inline alert variations because they’re not supported by PatternFly yet

Announcement banner

  • Deleted black Announcement banner because it clashes with navigation bar

Button

  • Deleted Large close buttons, aligning to using one close button like PatternFly

Call to action

  • Deleted Disabled Call to action because it’s not used anywhere
